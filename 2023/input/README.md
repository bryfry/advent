# Input Files

Filename format:

`{{ day }}-{{ type }}{{ part }}.txt`

Where:

 - day: `01`, `02`, `03` ... `25`
 - type: `e` for example or `p` for puzzel input type
 - part: `1` part 1, `2` part 2

Symlinks are encouraged when content doesnt change

### Examples:

```
01-e1.txt
01-p1.txt
01-e2.txt
01-p2.txt
...
09-e1.txt
09-p1.txt
09-e2.txt
09-p2.txt
...
25-e1.txt
25-p1.txt
25-e2.txt
25-p2.txt
```
